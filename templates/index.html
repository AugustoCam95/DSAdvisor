Welcome to the DSAdvisor !

DSAdvisor is a web tool to guide the user in the data processing steps for the correct pre-processing of the data in order to be used by machine learning models. This entire verification process occurs openly for the user along with tutoring on the concepts covered in each step of the proposed guideline, the tool seeks to bring users closer to the concepts of Machine Learning and Data Mining.

In this guideline, we have the following steps:

  • Upload data (only CSV files so far)
  • Descriptive statistics of the Data Set
  • Outliers Report
  • Normalization
  • Feature Selection
  • Resample Techniques
  • Run Models
  • Select Metrics
  • Results Representation
  • Guarantee Reproducibility
